# 📱 小红书一键下载快捷指令 — 完整教程

## 整体架构

```
小红书分享链接
    ↓
iOS 快捷指令（接收文本）
    ↓
调用解析 API（Vercel 免费部署）
    ↓
获取无水印图片/视频 URL
    ↓
保存到相册
```

---

## 第一步：部署解析 API（一次性）

### 方法 A：使用 Vercel（推荐，免费）

1. 注册 [vercel.com](https://vercel.com)（免费账号）
2. 安装 Vercel CLI：`npm i -g vercel`
3. 解压本包，进入文件夹：`cd xhs-parser`
4. 执行部署：`vercel --prod`
5. 记录部署后的域名，例如：`https://xhs-parser-xxx.vercel.app`

> ⚠️ 部署完成后，你的 API 地址为：  
> `https://你的域名.vercel.app/api/parse`

### 📋 快捷指令完整步骤

#### 动作 1：接收共享内容
- 动作名称：**「接收」**
- 配置：接收「文本」
- 勾选：「在共享表单中显示」「不显示时也进行」

---

#### 动作 2：获取剪贴板（备用）
- 动作名称：**「获取剪贴板」**
- 用途：如果用户直接运行快捷指令而没有共享文本

```
如果 [快捷指令输入] 没有值
    将剪贴板内容 赋值给 变量 「输入文本」
否则
    将快捷指令输入 赋值给 变量 「输入文本」
结束 如果
```

---

#### 动作 3：提取 URL
- 动作名称：**「从...获取 URL」**
- 输入：变量「输入文本」
- 将结果赋值给变量：**「小红书链接」**

---

#### 动作 4：获取网页内容（调用 API）
- 动作名称：**「获取 URL 的内容」**
- URL：
  ```
  https://你的域名.vercel.app/api/parse?url=【小红书链接变量】
  ```
- 方法：`GET`
- 将结果赋值给变量：**「API响应」**

---

#### 动作 5：解析 JSON
- 动作名称：**「获取...的词典值」**
- 输入：变量「API响应」
- 将结果赋值给变量：**「数据」**

---

#### 动作 6：获取类型
```
获取「数据」中键「type」的值 → 赋值给变量「类型」
获取「数据」中键「title」的值 → 赋值给变量「标题」
```

---

#### 动作 7：条件判断 — 视频或图片

```
如果 「类型」包含 「video」
    ↓（视频流程）
    获取「数据」中键「video」的值 → 赋值给「视频URL」
    获取「视频URL」的 URL 内容 → 赋值给「视频文件」
    将「视频文件」存储到相册
    显示通知：「✅ 视频已保存到相册！」
    
否则（图片流程）
    获取「数据」中键「images」的值 → 赋值给「图片列表」
    重复操作「图片列表」中的每一项
        获取当前 URL 的内容 → 赋值给「图片文件」
        将「图片文件」存储到相册
    结束重复
    显示通知：「✅ 图片已全部保存！共 X 张」
    
结束 如果
```

## 第三步：设置使用方式

### 方式 A：从小红书直接分享
1. 在小红书 App 内打开任意笔记
2. 点击右上角「···」→「分享」→「更多」
3. 在系统分享菜单找到你的快捷指令名称，点击

### 方式 B：复制分享文本后运行
1. 复制小红书的分享文本（如：`http://xhslink.com/xxx 复制后打开小红书...`）
2. 打开「快捷指令」App，运行本快捷指令
3. 自动从剪贴板读取链接并解析

## ⚠️ 注意事项

| 项目 | 说明 |
|------|------|
| 视频保存 | 视频文件较大，请保持网络连接 |
| 私密笔记 | 仅限公开笔记，私密内容无法解析 |
| API 限制 | Vercel 免费版每月 100GB 流量，个人使用完全够用 |
| 合规使用 | 仅供个人学习收藏，请勿二次传播 |
| 小红书更新 | 如解析失败，可能是 XHS 更改了接口，需更新 API 代码 |

---

## 🛠 备选方案：使用第三方解析服务

如果不想部署服务器，可以在快捷指令中直接调用以下公共 API（稳定性不保证）：

```
GET https://api.xhsdownload.com/parse?url=YOUR_URL
GET https://xhstools.com/api/parse?url=YOUR_URL
```

在快捷指令的「获取 URL 内容」动作中直接填入上述地址即可。

## 📞 问题排查

**Q: 快捷指令提示「无法获取内容」**  
A: 检查 API 地址是否正确，或网络是否正常

**Q: 解析失败**  
A: 小红书可能更新了接口，在 Vercel 重新部署最新代码

**Q: 视频只保存了封面**  
A: 部分视频需要登录才能获取原始流，可尝试在 API 中添加 Cookie
